I want you to create a number of tasks for code smells including DRY, magic numbers and refactoring large functions into smaller ones. ## Results

I want the result to be code that we can use in class and discuss to make better. Please also provide the expected results after the code smell or refactoring is done. Give me the code smell and the excepted result right after but with a little spacing, before the next code smell.
